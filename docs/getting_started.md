# Manual de Usuario CLI - Proyecto MOA

## Introducción

Este manual proporciona instrucciones detalladas sobre cómo utilizar los comandos `status`, `run` y `build-model` en el proyecto MOA. Estos comandos son esenciales para gestionar y ejecutar diferentes aspectos del proyecto.

## Comando: status

El comando `status` se utiliza para obtener información sobre el estado actual del proyecto, incluyendo la versión de los componentes utilizados y cualquier error o advertencia relevante.
