# Guía de Model Sculpting

## Introducción

La creación de modelfiles especializados es un proceso crucial en el desarrollo de aplicaciones basadas en Solaria. Este documento proporciona una guía paso a paso para crear modelfiles, utilizar el comando `build-model`, reducir latencia y ruido IA-IA.
